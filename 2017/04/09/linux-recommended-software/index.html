聊天软件

由于 QQ 并没有 Linux 的版本，当然你也可以选择用 Wine 来运行 QQ.exe 但是不推荐这么做。这里推荐几款在 Linux 下面好使的聊天软件。

  1. Wire，一款跨平台、界面优雅的聊天软件。 选择 Linux 的二进制下载，可以安装 .deb 也可以直接下载 AppImage 然后直接命令行运行。

  2. 飞鸽传书，一款跨平台的局域网聊天软件，可用来在局域网进行聊天和文件传输不用担心隐私泄露问题。

  3. irssi，一个终端的 IRC 协议聊天工具，可以和世界各国的朋友一起聊天。

办公软件

  1. 微软的 Office 也没有推出 Linux 的版本，国产的 WPS 是支持 Linux 的但是很久没有更新了而且不是很好使。这里推荐 Chrome 的 Web 应用程序 Office Online，请自备梯子如果没有梯子请参考我的博客 Lnux 翻墙终极版，Office 创建的文件是自动保存在 OneDrive 里面的，操作体验和 Office 2016 很像。

  2. typroa，书写 markdown 的神器，直接就预览了。

开发软件

  1. JetBrains系列，包含了常用的 Intellij IDEA,WebStorm 等等，跨平台就是好。该软件解压出来添加到环境变量中即可使用无须安装。

  2. Visual Studio Code，直接下载 .deb 或者 .rpm 包安装即可使用，微软出的代码编辑器配合插件特别好使。

  3. dbeaver,sql 客户端功能类似于 navicat。

  4. redis-desktop-manager，redis 的图形化客户端。-

效率软件

  1. FileZilla，一个 ftp/sftp 图形化管理工具。

  2. Nylas，一个漂亮的跨平台的邮箱软件。

影音软件

  1. 网易云音乐，Linux 下面少有的在线音乐播放器，而且颜值还这么高。

  2. mpv，一款影音播放器，界面很清爽而且可以很轻松配置成无边框配置请参考mpv教程

  3. kodi，支持自定义源代的在线视频播放器，功能特别强大，
    推荐配置 https://github.com/taxigps/xbmc-addons-chinese

小工具

  1. Albert，一款类似于 Mac 的 Spotlight 工具特别好使。

  2. Redshift，根据时间自动调节显示器颜色（防蓝光）的软件，配置文件redshift.conf替换 ~/.config/redshift.conf。

  3. cliplt，一款剪切板管理软件，可以看到历史的剪切记录。

总结

Linux 的软件生态系统还是满丰富的，如果你有什么值得推荐的软件请在下面留言吧，谢谢。
